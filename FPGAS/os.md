## Instalación en Linux

Como mencioné antes, Linux destaca por ser mucho más rápido a la hora de ejecutar programas pero es un poco más complicado de usar. Principalmente las instalaciones para las distribuciones más usadas [(Linux Mint, Ubuntu, Arch Linux)](#referencias-de-instalaciones-de-linux).

> En caso de querer usar Linux y no tenerlo. Mi recomendación es usar **Linux Mint** siendo el más familiar a Windows.

*Pd: Hay varias formas de instalar Linux pero la más recomendada y usada para estos proyectos es tener los dos sistemas operativos, a esto se le conoce como **dual boot**, es la mejor forma para no tener que eliminar la partición de Windows. Otra forma es hacerlo en un disco aparte.*

## Instalación en Windows

Acá no hay mucho que decir, uno de los sistemas operativos más usados pero últimamente anda muy mal y es de pago. En caso de que borren por "*accidente*" la partición de Windows les recomiendo reinstalarlo de esta forma (Muy despistados si lo hacen jsjsjs pero a mí me pasó una vez):

- **Instalación de Windows 11**
  - [Instalación en Inglés](https://www.youtube.com/watch?v=d3GqcVrSIT0)
  - [Instalación en Español](https://www.youtube.com/watch?v=Rj7YDLwWY6A)

Acceder a la bios:
> Para acceder a la BIOS depende del computador, hay dos formas: desde Windows o con una tecla (depende de su computador) al reiniciar este. Si es por Windows es: settings -> update and security -> recovery -> advanced startup -> restart now.

Otra forma:
> Si es por tecla, al reiniciar el computador hay que presionar la tecla indicada (toca buscar la referencia en internet) para acceder a la BIOS.
